## 🚀 About The Project

**FunFair** is a full-stack multi-vendor e-commerce platform built to simulate a real-world marketplace system.  
It allows vendors to register, manage products, handle orders, and update order status.  
Customers can browse products, add items to cart, complete checkout, and view their order details.

This project helped me improve my skills in:
- full-stack development
- authentication and authorization
- REST API integration
- MongoDB database handling
- building real-world project workflows

---

## ✨ Key Features

### 👨‍💼 Vendor Side
- Vendor registration and login
- Secure JWT authentication
- Vendor dashboard
- Add new products
- Edit and delete products
- Manage stock and pricing
- View customer orders
- Update order status

### 🛍️ Customer Side
- Browse products
- Add products to cart
- Checkout system
- Place orders
- View order details

### ⚙️ System Features
- Full-stack architecture
- REST API integration
- MongoDB database storage
- Protected routes
- Organized backend structure
- Order management workflow
